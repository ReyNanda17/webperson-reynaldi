Tentang Saya

Saya adalah seorang mahasiswa Teknik Informatika dengan minat khusus dalam pengembangan web dan kecerdasan buatan. Saat ini, saya sedang menempuh pendidikan di Universitas Respati Yogyakarta dan telah berpartisipasi dalam beberapa proyek terkait teknologi informasi, termasuk pengembangan aplikasi web responsif dan penelitian kecerdasan buatan.

Saya belajar untuk memperluas wawasan dan keterampilan praktis. Selain itu, saya belajar bekerja dengan berbagai alat dan bahasa pemrograman, seperti HTML, CSS, JavaScript, dan Python, serta memiliki ketertarikan besar dalam pengembangan perangkat lunak berbasis AI.

Cita-cita saya adalah menjadi pengembang perangkat lunak profesional dan berkontribusi dalam menciptakan solusi teknologi inovatif yang dapat memajukan dunia digital. Saya percaya kombinasi antara keterampilan teknis, ketekunan dalam belajar, dan semangat kerja keras akan membawa saya mencapai kesuksesan di bidang teknologi.
